# Rotina de Kaprekar

Este programa implementa a Rotina de Kaprekar para números de quatro dígitos, encontrando a constante 6174.

## Como funciona
O usuário informa um número de quatro dígitos.  
O programa organiza os dígitos em ordem crescente e decrescente, faz a subtração e repete o processo até chegar na constante 6174.

## Como executar
Execute o código em Python e digite um número de quatro dígitos quando solicitado.
